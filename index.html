HEADER

The <header> HTML5 element defines a header of your webpage. It is always visible for the users and is located at the top of your website

Example :

Code:

<header>
<h1>JavaScript</h1>
<h3>What is JavaScript?</h3>
<p>Today we are going to talk about JavaScript</p>
</header>

OUTPUT:

JavaScript

What is JavaScript?

Today we are going to talk about JavaScript


NAV

<nav> is one of the main HTML5 semantic tags. It depicts the space for navigation links on your website.

Example :

Code:

<nav>
<a ref="#gamified">Gamified Courses</a> /
<a ref="#tutorials">Tutorials</a> /
<a ref="#doggo">Space doggo courses</a>
</nav>

OUTPUT:

Gamified Courses / Tutorials / Space doggo courses

SECTION

The <section> HTML5 semantic tag defines a separate section within a webpage, with its own content.

Example :

Code:

<section>
<h1>Section Heading</h1>
<p>The section tag can contain any elements</p>
</section>

OUTPUT:

Section Heading

The section tag can contain any elements.


ARTICLE

The <article> element is one of the major HTML5 semantic tags. It is used to define the article content on your website. It is usually used for large parts of text.

Example :

Code:

<article>
<h1>Mozilla Firefox</h1>
<p>Firefox is a free, open-source web browser from Mozilla, released in 2004.</p>
</article>

OUTPUT:

Mozilla Firefox

Firefox is a free, open-source web browser from Mozilla, released in 2004.

MAIN

<main> is one of the HTML5 semantic tags, which sets space for the main (most important) content of a webpage. It can be an article, regular paragraph or anything else.

Example :

Code:

<main>
<h1>Web Browsers</h1>
<p>Google Chrome, Firefox, and Internet Explorer are the most used browsers today.</p>
<article>
<h1>Google Chrome</h1>
<p>Google Chrome is a free, open-source web browser developed by Google, released in 2008.</p>
<p>><strong>Note:</strong>The main tag is not supported in internet Explorer and earlier version.</p>
</article>
</main>

OUTPUT:

Web Browsers

Google Chrome, Firefox, and Internet Explorer are the most used browsers today.

Google Chrome

Google Chrome is a free, open-source web browser developed by Google, released in 2008.

Note: The main tag is not supported in internet Explorer and earlier version.
